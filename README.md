# Bank Management System

## Tech Stack
- React.js (Frontend)
- ASP.NET Core Web API (Backend)
- SQL Server (Database)
- Entity Framework Core
- JWT Authentication

## Features
- User Registration & Login
- Deposit / Withdraw / Transfer
- Transaction History
- Profile Management
- Admin Panel (User & Account Control)
- Account Status (Active / Hold / Blocked)

## How to Run

Backend:
dotnet run

Frontend:
npm install
npm run dev
